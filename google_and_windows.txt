Google ===========================

Select previous tab - Alt + H
Select next tab - Alt + L
Move current tab left - Alt + J
Move current tab right - Alt + K
Close current tab - Alt + I
New tab - Alt + N
Duplicate current tab - Alt + O
Go back to the previous page - Alt + Comma
Go forward to the next page - Alt + Period
Pin / Unpin current tab - Alt + P
Reload tab - Alt + U

hide all tabs - Windows + d
close program - Alt + F4

full screen for tab - Windows + Up
